# SentinelAI - AI Smart Home Security System

An AI-powered smart home security system built using Computer Vision, Artificial Intelligence, and IoT.

## Features

- Real-time Person Detection using YOLOv8
- Face Recognition using InsightFace
- Unknown Visitor Detection
- Visitor Logging
- Telegram Alerts (Coming Soon)
- Flask Dashboard (Coming Soon)
- ESP32 Motion Detection (Coming Soon)

## Tech Stack

- Python
- OpenCV
- YOLOv8
- InsightFace
- Flask
- SQLite
- ESP32
- Git & GitHub
